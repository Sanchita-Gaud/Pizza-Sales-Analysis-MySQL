# Pizza-Sales-Analysis-MySQL
🍕 Pizza Store Sales Analysis using MySQL
📌 Project Overview

This project analyzes pizza store sales data using MySQL to answer important business questions and uncover actionable insights.

Using a real-world dataset from Kaggle, I explored customer ordering behavior, sales performance, product popularity, and revenue trends by writing SQL queries involving joins, aggregate functions, subqueries, and window functions.

The objective of this project was to strengthen SQL skills while solving practical business problems through data analysis.

🎯 Objectives
Analyze overall sales performance
Identify customer ordering patterns
Find the best and worst selling pizzas
Calculate revenue generated
Compare pizza categories and sizes
Rank products based on sales performance
🛠️ Tools Used
MySQL
SQL
Kaggle Dataset
📂 Dataset

The project uses the Pizza Sales dataset from Kaggle.

The dataset consists of multiple related tables:

Orders
Order Details
Pizzas
Pizza Types
📊 Business Questions Solved
Retrieve the total number of orders placed.
Calculate the total revenue generated.
Identify the highest-priced pizza.
Find the most common pizza size ordered.
List the top 5 most ordered pizza types.
Calculate the total quantity ordered for each pizza category.
Determine the distribution of orders by hour.
Find the category-wise distribution of pizzas.
Calculate the average number of pizzas ordered per day.
Determine the top 3 pizza types by revenue for each category.
